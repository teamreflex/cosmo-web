This is a [Subsquid](https://docs.sqd.ai/sdk/overview/) blockchain indexer that watches the [Abstract blockchain](https://www.abs.xyz/) and scrapes the COSMO API for objekt metadata.

The following entities are tracked:

- objekt collections (the type of objekt)
- objekt tokens (each unique token and ownership)
- objekt transfer
- objekt transferability changes
- COMO balance changes
- gravity votes
